REQ-001 — Score 100 — OK

Texte exigence :

The braking system shall stop the train within 800 m when the emergency brake is applied at 160 km/h on dry track.

Issues détectées :

Aucune issue détectée.

Suggestions IA : IA

  • Reformulate requirement: At maximum operational mass, on straight, level track (|grade| ≤ 0.5‰) and dry rail (no visible moisture, RH ≤ 80%, no precipitation during test), the train shall stop within 800 m when an emergency brake command is issued at 160.0 ± 1.0 km/h; stopping distance measured from brake-command reception to zero speed.
  • Acceptance criteria: 20 consecutive runs; initial speed 160.0 ± 1.0 km/h; train mass ≥ 98% of maximum operational mass; |grade| ≤ 0.5‰; curve radius ≥ 2000 m; wind ≤ 5 m/s; dry rail = no visible moisture, RH ≤ 80%, no precipitation; distance measurement accuracy ≤ ±2 m. Pass if ≥ 19/20 runs ≤ 800 m and no run > 820 m.
  • Alternative acceptance: 30 runs under the defined dry, level, straight-track conditions; demonstrate with 95% confidence that the true 95th-percentile stopping distance ≤ 800 m; no individual run > 820 m.
REQ-002 — Score 70 — À risque

Texte exigence :

The HMI should display the status quickly and clearly for the driver.

Issues détectées :

  • MINOR — Modal verb 'should' detected (weak commitment). Prefer 'shall' or measurable phrasing.
  • MINOR — Ambiguous term 'quick' detected (not measurable).
  • MINOR — Acceptance criteria missing (verification might be unclear).

Suggestions IA : IA

  • Remplacer les modaux faibles (should/may/…) par une formulation normative mesurable (shall + métriques).
  • Remplacer les termes qualitatifs par des critères quantifiés (temps, taux, seuils, tolérances).
  • Ajouter des critères d’acceptation concrets (Given/When/Then, seuils, tolérances).
  • Replace with: 'The HMI shall display the current train status within 1.0 s of receiving a valid status-change message from the control system.'
  • Add clarity requirement: 'The HMI shall present the status on the primary screen without user input, using text character height >= 4 mm and contrast ratio >= 4.5:1, with a status icon and label.'
  • Acceptance criteria: (1) In 30 simulated status changes, >=95% are displayed within 1.0 s; (2) Text character height measured >= 4 mm; (3) Contrast ratio measured >= 4.5:1; (4) Status icon and label visible on the primary screen without user input.
REQ-003 — Score 100 — OK

Texte exigence :

The infusion pump shall deliver 5 mL/h ± 5% for flow rates between 1 mL/h and 10 mL/h at 23°C with saline.

Issues détectées :

Aucune issue détectée.

Suggestions IA : IA

  • Requirement: At 23°C ±2°C using 0.9% saline and outlet pressure 0 ±1 kPa, the pump shall deliver an average flow within ±5% of the commanded setpoint for any setpoint from 1.0 to 10.0 mL/h, measured over 60 min.
  • Acceptance: Verify at setpoints {1, 2, 5, 8, 10} mL/h; at 23°C ±2°C with 0.9% saline and outlet pressure 0 ±1 kPa, each of n=10 production units shall deliver an average flow within ±5% of setpoint over 60 min; all units must pass.
  • Acceptance: For setpoint 5.0 mL/h at 23°C ±2°C with 0.9% saline and outlet pressure 0 ±1 kPa, using a calibrated gravimetric method with uncertainty ≤1% of reading, the delivered volume in 60 min shall be 4.75–5.25 mL.
REQ-004 — Score 90 — À risque

Texte exigence :

The ECU shall provide diagnostic information if needed.

Issues détectées :

  • MINOR — Ambiguous term 'if needed' detected (not measurable).

Suggestions IA : IA

  • Remplacer les termes qualitatifs par des critères quantifiés (temps, taux, seuils, tolérances).
  • Rephrase requirement: The ECU shall provide diagnostic information when any DTC is set or cleared and upon receipt of a diagnostic read request on the diagnostic communication interface.
  • Acceptance criterion: Following a diagnostic read request, the ECU shall return a single response containing DTC code, timestamp, and current operating mode within 200 ms.
  • Acceptance criterion: Timestamp is milliseconds since ignition ON (resolution >= 1 ms, accuracy +/- 10 ms); DTC code is a 3-byte value; operating mode is a 1-byte numeric code; all fields present in the same response.
