REQ-001 — Score 100 — OK

Texte exigence :

The braking system shall stop the train within 800 m when the emergency brake is applied at 160 km/h on dry track.

Issues détectées :

Aucune issue détectée.

Suggestions IA : AI

  • Refine requirement: Stop distance ≤ 800 m from emergency brake command at 160.0 ± 1.0 km/h on dry, straight (R ≥ 1000 m), level (|grade| ≤ 2 ‰) track with train at maximum operating mass; distance measured from command to standstill.
  • Acceptance criteria: 10 consecutive emergency stops under the above conditions; each run: initial speed 160 ± 1 km/h, stopping distance ≤ 800 m, no precipitation in prior 60 min, crosswind ≤ 5 m/s.
  • Acceptance criteria (measurement): Use calibrated distance measurement with absolute error ≤ 0.5% or 5 m (whichever greater); synchronize brake command timestamp and distance measurement within ±50 ms.

The HMI should display the status quickly and clearly for the driver.

Issues détectées :

  • [MINOR] Modal verb 'should' detected (weak commitment). Prefer 'shall' or measurable phrasing.

  • [MINOR] Acceptance criteria missing (verification might be unclear).

Suggestions IA : AI

  • Remplacer les modaux faibles (should/may/…) par une formulation normative mesurable (shall + métriques).
  • Remplacer les termes qualitatifs par des critères quantifiés (temps, taux, seuils, tolérances).
  • Ajouter des critères d’acceptation concrets (Given/When/Then, seuils, tolérances).

  • Add AC: 'A persistent status area shall be visible on the main driving screen without navigation; each critical status icon shall be ≥10 mm and accompanied by a text label.'
REQ-003 — Score 100 — OK

Texte exigence :

The infusion pump shall deliver 5 mL/h ± 5% for flow rates between 1 mL/h and 10 mL/h at 23°C with saline.

Issues détectées :

Aucune issue détectée.

Suggestions IA : AI

  • REQ: For setpoints 1.0–10.0 mL/h, at 23°C ±1°C using 0.9% saline, the pump shall deliver an average volumetric flow within ±5.0% of the setpoint when averaged over 60 minutes after a 5-minute stabilization.
  • AC: Test at 1.0, 5.0, and 10.0 mL/h; exclude first 5 min; measure for 60 min gravimetrically with scale resolution ≤0.01 g and uncertainty ≤0.1% of reading; pass if each average is within ±5% of setpoint.
  • AC: Use standard infusion set with new tubing; maintain outlet backpressure 100 ±10 mmHg; ambient 23°C ±1°C; fluid 0.9% saline; document density used for mass-to-volume conversion.
REQ-004 — Score 90 — À risque

Texte exigence :

The ECU shall provide diagnostic information if needed.

Issues détectées :

  • [MINOR] Ambiguous term 'if needed' detected (not measurable).

Suggestions IA : AI

  • Remplacer les termes qualitatifs par des critères quantifiés (temps, taux, seuils, tolérances).
  • Rewrite requirement: 'The ECU shall provide diagnostic information (DTC code, timestamp, current operating mode) upon receipt of a valid diagnostic request on the diagnostics interface.'
  • Alternative trigger: 'The ECU shall record diagnostic information (DTC code, timestamp, current operating mode) when a DTC is set and make it retrievable via the diagnostics interface.'
  • Add acceptance criteria: 'Given ≥1 stored DTC, a valid diagnostic request returns at least one record with DTC code, timestamp, and current operating mode; given 0 stored DTCs, the ECU returns an explicit no-DTC indicator; given an invalid request, the ECU returns an error response.'
